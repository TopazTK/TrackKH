# Auto-Tracker for KH1 Randomizer

This is a full-fledged Auo-Tracker for Denhonator's KH1 Randomizer, heavily inspired by [zax's web-tracker](https://tracker.zaxu.xyz/).

## Features
- Automatic and Manual tracking.
- Item exclusion from tracking logic.
- Ability to resize the window at will.
- Background color selection to allow Chroma Keying.
- Icon style switching between Classic and Simplified.
- Many more to come!

---

## Checks
- All Magic and their appropriate levels.
- All redeemed Summons.
- All learned Trinities.
- All finish-related items.
- Reports and Jack-in-the-Box.
- All rescued Puppies and redeemed Gifts.
- All obtained and sent Postcards.

---

## Planned Features
- Fullscreen Overlaying to KH, with free movement of all icons.
- Android Support (Both Manual and Automatic).
- Check modification and distinction options.
